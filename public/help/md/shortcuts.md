# Shortcuts

The following keyboard shortcuts are supported within the application.

|     Windows      | macOS/Linux | Action                  |
| :--------------: | :---------: | :---------------------- |
|  Alt + Ctrl + N  |    ⌘ ^ N    | Create new analysis     |
|     Ctrl + S     |     ⌘ S     | Save current analysis   |
|     Ctrl + L     |     ⌘ L     | Load analysis from file |
| Ctrl + Shift + S |    ⇧ ⌘ S    | Create bookmark         |
| Ctrl + Shift + L |    ⇧ ⌘ L    | Show bookmarks          |
|     Ctrl + I     |     ⌘ I     | Import data             |
|     Ctrl + K     |     ⌘ K     | Open data filter        |
|     Ctrl + U     |     ⌘ U     | Show Recent Analysis    |
|     Ctrl + /     |     ⌘ /     | Show/Hide Sidebar       |
| Ctrl + Shift + D |    ⇧ ⌘ D    | Show Dashboard          |
| Ctrl + Shift + P |    ⇧ ⌘ P    | Show Plots              |
| Ctrl + Shift + M |    ⇧ ⌘ M    | Show Map                |
| Ctrl + Shift + X |    ⇧ ⌘ X    | Show Spreadsheet        |
|  Alt + Ctrl + E  |    ⌘ ^ E    | Export analysis         |

> [!warning] Some browser shortcuts might be overwriten by the application
