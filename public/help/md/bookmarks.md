# Bookmarks

Bookmarks allow you to save snapshots of your analysis state, making it easy to compare different stages of your analysis or return to previous configurations.

## Using Bookmarks

You can create a bookmark in two ways:

- Click the ⋮ button in the Sidebar menu and select Save Bookmark…
- Use the keyboard shortcut:
  - Windows: <kbd><i class="bi bi-windows"></i> + Shift + S</kbd>
  - macOS: <kbd>⇧ ⌘ S</kbd>

Access your bookmarks through:

- Sidebar menu ⋮ → Show Bookmarks
- Keyboard shortcut:
  - Windows: <kbd><i class="bi bi-windows"></i> + Shift + L</kbd>
  - macOS: <kbd>⇧ ⌘ S</kbd>

## Saved State Information

Each bookmark captures:

- Active filters and thresholds
- Data subset configurations
- Map and graph settings
- Dashboard layout and panel settings
- Color schemes and visual preferences

> [!note]- title
> All bookmarks are automatically saved when you save your analysis file

### Best Practices

- Add bookmarks before making significant changes to your analysis
- Use bookmarks to compare different analysis approaches
- Consider creating bookmarks for:
  - Initial data state
  - Important findings
  - Different filter combinations
  - Alternative visualization settings
