# Flag Data

When analyzing data, some datasets might have issues requiring to exclude them from the analysis. Flagging data allows to select individual datasets (corresponding to a row in the spreadsheet) and remove them from the analysis, without removing them entirely. A paramter as well as a reason for excluding the data can be selected.

## Flagging Data

Data can be flagged by selecting single or groups of data points or markers from the graph and map view.
