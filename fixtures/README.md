# Test Data

This folder contains subfolders with test data for unit testing of individual statistical and cluster functions.
The `.json` files contain all tests from their corresponding subfolders, including results from their corresponding Python implementations for comparison.

```bash
.
├── test_data_cluster       # Test data for cluster functions
├── test_data_stats         # Test data for statistical functions
├── test_data_cluster.json  # Tests with results from Python implementation
└── test_data_stats.json    # Tests with results from Python implementation
```
